#### 多个不同的数组沿着不同的轴拼接在一起

    import numpy as np
    a = np.arange(4).reshape((2, 2))

    a
    array([[0, 1],
       [2, 3]])

    b = np.array([[4, 5],[6, 7]])

    b
    array([[4, 5],
       [6, 7]])

    a和b都是2x2的矩阵
    将a和b在垂直方向上合并成4x1矩阵
    np.vstack((a, b))
    array([[0, 1],
       [2, 3],
       [4, 5],
       [6, 7]])

    将a和b在垂直方向上合并成2x4矩阵
    np.hstack((a, b))
    array([[0, 1, 4, 5],
       [2, 3, 6, 7]])

    如果我们有个一维向量怎么合并到二维矩阵上
    v = np.array([8, 9])

    将该向量在垂直方向合并到a上
    np.vstack((a, v))
    array([[0, 1],
       [2, 3],
       [8, 9]])

    将该向量在水平方向上合并到a上，发现报错，原因是两个数组具有不同的维度
